Este é o compilador implementado para a Unidade Curricular Compiladores.

Para compilar digite no terminal o seguinte comando:
	make

Para realizar um teste com uma entrada digite:
	./tiny <nome-do-arquivo>.cm


***Talvez seja necessário atualizar o path em binary.c para salvar o binário.
